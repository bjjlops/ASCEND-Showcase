# Screenshots manifest

Product screenshots for the showcase. `✅ in` = present and wired into
[`docs/gallery.md`](../../docs/gallery.md); `⬜ needed` = still to capture.

## Guidance

- **Format:** PNG preferred. Use the dark theme — it's the ASCEND command-center look.
- **Width:** ~1600px for full-window shots; crop out the browser chrome and OS bar.
- **Content:** use demo/seed data — **no real personal data, no real account balances, no broker
  credentials, no API keys** visible in frame. *(The provided Planner shot had a personal email in
  the account corner; it was redacted to `you@ascend.app` before wiring in.)*
